Python2 特点介绍

1.中文支持

python3中，是默认支持中文的，但是在python2中就需要一定的额外操作才可以

python2中，如果直接在程序中用到了中文，比如

print('你好')

如果直接运行输出，程序会出错：

解决的办法为：在程序的开头写入如下代码，这就是中文注释

#coding=utf-8

修改之后的程序:

#coding=utf-8
print('你好')

运行结果:

你好

注意：

在python的语法规范中推荐使用的方式：

# -*- coding:utf-8 -*-

2.输入

2.1 raw_input()

在Python2中，raw_input 和 Python3 的 input 用法完全相同

2.2 input()

input()函数与raw_input()类似，但其接受的用户输入必须是表达式。

>>> a = input() 
123
>>> a
123
>>> type(a)
<type 'int'>
>>> a = input()
abc
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
  File "<string>", line 1, in <module>
NameError: name 'abc' is not defined
>>> a = input()
"abc"
>>> a
'abc'
>>> type(a)
<type 'str'>
>>> a = input()
1+3
>>> a
4
>>> a = input()
"abc"+"def"
>>> a
'abcdef'
>>> value = 100
>>> a = input()
value
>>> a
100

input()接受表达式输入，并把表达式的结果赋值给等号左边的变量

3.输出

在 Python 的输出，除了使用支持 print(xxx), 还支持 print xxx.

比如：

print('Hello World!')  # Python2 和 Python3 都支持
print 'Hello World!'  # 只有 Python2 支持

结果为

Hello World!
Hello World!

4.字典键的判断

在 Python2 中，除了可以使用 in 来判断字典是否存在指定的键，还可以使用 has_key

>>> a = {'age':18}
>>> 'age' in a  # Python2 和 Python3 都支持
True
>>> a.has_key('age)  # 只有 Python2 支持
True

5.经典类和新式类

在 Python3 里如果一个类没有父类则默认是继承 object 类。

在 Python2 里类分为经典类和新式类，如果一个类写明继承 object 则是新式类，类的使用方式和 Python3 相同。如果定义类时没有写明父类，则是经典类。

经典类的特点已经不用研究，因为经典类在公司里已经不再使用了。有兴趣可以自行研究，课程里不会进行讲解。

也就是说，如果 Python2 和 Python3 要具备相同的类特征，则 Python2 必须需要写明继承 object。
