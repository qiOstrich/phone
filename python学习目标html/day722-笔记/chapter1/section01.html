第五天

字符串输出

demo

    name = 'chris'
    job = '讲师'
    address = '上海市'

    print('--------------------------------------------------')
    print("姓名：%s" % name)
    print("职位：%s" % job)
    print("公司地址：%s" % address)
    print('--------------------------------------------------')

结果:

    --------------------------------------------------
    姓名： chris
    职位： 讲师
    公司地址： 上海市
    --------------------------------------------------
