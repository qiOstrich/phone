嵌套循环

前面学习过if的嵌套了，想一想if嵌套是什么样子的？ 类似if的嵌套，while嵌套就是：while里面还有while

while嵌套的格式

while 条件1:
        外循环代码块
        while 条件2:
            内循环代码块

内循环一旦开始，则要执行到条件2不满足，才会继续执行外循环的代码。

i = 0  # 内外循环的控制变量不能一样
while i < 5:

    j = 0  # 内循环的控制变量必须要在外循环里初始化
    while j < 5:
        print('j= %d' % j)
        j += 1

    print('i= %d' % i)
    i += 1
