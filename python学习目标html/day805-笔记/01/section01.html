Base64编码

Base64是一种用64个字符来表示任意二进制数据的方法，可以称为密码学的基石。可以将任意的二进制数据进行Base64编码，所有的数据都能被编码为并只用65个字符就能表示的文本文件。

原理(了解)

转码的核心在于： 3*8=4*6

  • 要转化的字符串能被3整除

    要对字符串 'abc' 进行Base64加密，需要先拿到 abc 的二进制。abc 转换成为字节分别对应 97,98,99

    转换成为二进制，八个一位： 01100001 01100010 01100011

    对二进制重新组合，6个一组：011000 010110 001001 100011

    转换成为十进制: 24 22 9 35

    对照Base64码表: YWJj

    所以，字符串 abc 进行64为加密以后就是 YWJj

  • 要转换的字符串除以3余2

    将字符串 ab 进行Base64加密

    转换成为二进制，八个一位： 01100001 01100010

    对二进制重新组合，6个一组: 011000 010110 001000 最后只剩下四位，需要在最后补0

    转换成为十进制： 24 22 8

    对照Base64码表： YWI= 使用一个 = 来表示，最后面补了两个0

  • 要转化的字符串除以3余1

    将字符串 a 进行 Base64加密

    转换成为二进制，八个一位： 01100001

    对二进制重新组合，6个一组: 011000 010000 最后只剩下两位，需要在后面补四个0

    转换成为十进制： 24 16

    对照Base64码表： YQ== 使用两个 = 来表示，最后面补了四个0

Python中实现Base64编码

Python中的加密算法，都是先将字符串转换成为二进制，然后再对二进制进行加密。在进行base64编码前，我们也需要先将字符串转换成为Bytes,然后再进行编码。

将字符串和Bytes互相转换可以使用encode()和decode()方法。如下所示：

str1 = '你好'
bs = str1.encode() # 将字符串转换成为二进制列表
print(bs) # b'\xe4\xbd\xa0\xe5\xa5\xbd'
print(type(bs)) # bytes

bs.decode()  # 将二进制列表转换成为字符串

对Bytes进行Base64编码:

import base64

str1 = '你好'
str_bytes = str1.encode()  # 将字符串转换成为bytes
b64_value = base64.b64encode() # 将bytes进行bae64编码
print(b64_value)
value = base64.b64decode(b64_value)  # 将base64编码后的字符串进行解码，得到bytes
print(value.decode()) # 将bytes解码成为字符串

Base64严格意义上来说，只是一种编码格式，算不上是加密算法，因为它的编码格式是公开可逆的。
